# MediTrust - Doctor Appointment System

MediTrust is a full-stack doctor appointment booking system built using the MERN stack. It enables patients to book appointments, doctors to manage schedules, and admins to control the platform with secure authentication and role-based access.

---

## Features

- JWT-based Authentication (Login / Signup)
- Role-Based Access Control (Admin / Doctor / Patient)
- Book & Manage Appointments
- Doctor Dashboard (availability, approvals, earnings)
- Admin Panel (manage doctors, users, appointments)
- RESTful APIs
- Responsive UI

---

## Tech Stack

**Frontend:** React.js, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Authentication:** JWT  

---

## Architecture

- Routes → API endpoints  
- Controllers → Business logic  
- Models → Database schemas  

Frontend communicates with backend using REST APIs.
